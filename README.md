# cwebp-gui
GUI to batch-compile images to webp

## How to run

Prerequisites: <a href="Download Python 3.11.0">Install Python</a>

```
git clone https://github.com/philparzer/cwebp-gui
```

```
cd cwebp-gui
```

```
pip install PySimpleGUI
```


```
pip install webptools
```


```
python main.py
```
